# Practical-5: Jetpack Compose Authentication App with Navigation  
Mobile Application Development – Practical 5

This project demonstrates a complete **Login–Registration Authentication Flow** built entirely using **Jetpack Compose** and **Navigation Compose**, following modern Android development practices.

---

##  Project Overview  
This Android application showcases:

- User Authentication UI (Login + Registration)
- Navigation using Jetpack Compose Navigation
- Material 3 based modern UI
- Custom reusable form components
- State management using `remember` & `mutableStateOf`

---

##  Authentication Flow

### ** Login Screen**
- Email field  
- Password field  
- “Forgot Password?” text  
- Login Button  
- Navigation link to Registration  

### ** Registration Screen**
Includes input fields for:  
- Name  
- Phone  
- City  
- Email  
- Password  
- Confirm Password  
- Register Button  
- Navigation back to Login  

---

##  UI Components

### **Custom Form Fields**
- Reusable `TextField` components  
- Numeric-only input validation  
- Password input with secure visual transformation  

### **Material Design 3**
- Elevated card-style form containers  
- Modern typography & spacing  
- Guni Pink branding logo integrated in screens  

---

##  Technical Implementation

### **Jetpack Compose**
Modern declarative UI framework used for the full app.

### **Navigation Compose**
Handles screen transitions between:
- `login`
- `register`

### State Management
Implemented using:
- `remember`
- `mutableStateOf`
